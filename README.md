# Mohamad Diniy Iqbal - Professional Portfolio Website

Welcome to my professional portfolio website! This repository contains a modern, responsive, and interactive single-page portfolio designed to showcase my academic qualifications, technical skills, academic projects, and professional internship experience in Information System Management.

## 💻 Tech Stack
*   **Structure:** HTML5 (Semantic markup)
*   **Styling:** CSS3 (Custom design system, variables, ambient glow effects, responsive layouts)
*   **Interactivity:** Vanilla JavaScript (ES6+)
*   **Icons:** Lucide Icons (CDN)
*   **Typography:** Google Fonts (Outfit & Inter)

---

## ✨ Features
1.  **Ambient Glow Design:** A premium, modern visual aesthetic featuring soft background gradients and a clean glassmorphism dark/light theme.
2.  **Theme Toggle:** Persistent light/dark mode based on user preference, saved via local storage.
3.  **Dynamic Animations:** Scroll-reveal behaviors for sections and custom animated skill bars.
4.  **Academic Projects Showcase:** Detailed highlights of 6 key projects, from full-stack web applications to statistical data dashboards.
5.  **Interactive Contact Form:** Complete with visual submit states and toast confirmation alerts.
6.  **Print-Friendly Resume Mode:** Custom CSS print styles that automatically format the portfolio into a clean, professional PDF resume when printing or saving from the browser.

## 🛠️ How to Run Locally
1.  Clone this repository:
    ```bash
    git clone https://github.com/your-username/portfolio.git
    ```
2.  Navigate to the directory:
    ```bash
    cd portfolio
    ```
3.  Open `index.html` in any web browser, or serve it using local hosting environments like XAMPP, Live Server (VS Code), or python http server:
    ```bash
    python -m http.server 8000
    ```

## 🌐 Deploying to GitHub Pages
To publish your portfolio online for free:
1.  Upload this folder to a new public GitHub repository.
2.  Go to the repository **Settings** on GitHub.
3.  Navigate to **Pages** in the left sidebar.
4.  Under **Build and deployment**, select **Deploy from a branch**.
5.  Set the branch to `main` (or `master`) and directory to `/ (root)`, then click **Save**.
6.  Your site will be live within a few minutes!
